# 💅 The Nail Shop — Sistem Jadwal Karyawan

## Cara Deploy ke Vercel (GRATIS)

### Langkah 1 — Buat akun GitHub
- Buka github.com → Sign up (gratis)

### Langkah 2 — Upload project ini
- Buat repository baru di GitHub
- Upload semua file dari folder ini

### Langkah 3 — Deploy ke Vercel
- Buka vercel.com → Sign up with GitHub
- Klik "New Project" → Import dari GitHub
- Pilih repository thenailshop
- Klik Deploy → SELESAI!

### Langkah 4 — Share link
- Vercel akan kasih link seperti: thenailshop.vercel.app
- Share ke semua karyawan via WhatsApp

## Login
- Admin: admin / admin123
- Karyawan: nama (huruf kecil) / 1234
  Contoh: alfa / 1234
